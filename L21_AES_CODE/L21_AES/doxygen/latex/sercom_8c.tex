Set G\+C\+L\+K channel to generator. 

This will set the appropriate G\+C\+L\+K channel to the requested G\+C\+L\+K generator. This will set the generator for all S\+E\+R\+C\+O\+M instances, and the user will thus only be able to set the same generator that has previously been set, if any.

After the generator has been set the first time, the generator can be changed using the {\ttfamily force\+\_\+change} flag.
